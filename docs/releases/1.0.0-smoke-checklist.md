# 1.0.0 Smoke Checklist

Versions: `1.0.0-alpha.x`, `1.0.0`

## Tested on

- OS:
- Shell:
- Git:
- Node:

## 1) Root-safe read-only stays in repo root

- [ ] Read-only commands run in repo root.
- [ ] No worktree bind/provision occurs for read-only flow.

## 2) First mutating call provisions/binds worktree

- [ ] First mutating action provisions/binds a worktree.
- [ ] Mutating action executes in bound worktree.

## 3) Task delegation binds before execution and reuse across planner/implementer/reviewer

- [ ] Delegated task binds worktree before execution.
- [ ] Same binding is reused across planner, implementer, reviewer.

## 4) Path/workdir rewriting rules

- [ ] Repo-scoped absolute paths are rewritten to bound worktree.
- [ ] External absolute paths are not rewritten.
- [ ] Ambiguous paths block execution.

## 5) Fail-closed + explicit override behavior

- [ ] Unsafe/unclear state fails closed by default.
- [ ] Explicit override is required and honored when provided.

## 6) Cleanup advisory-only and appears only at defined moments

- [ ] Cleanup is advisory-only (no implicit destructive action).
- [ ] Cleanup prompt appears only at defined moments.

## How to record results

- Create one GitHub issue per alpha named `1.0.0-alpha.N smoke test results`.
- Paste this checklist into the issue and mark outcomes.
